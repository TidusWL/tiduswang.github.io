# 概率论在竞技博弈中的应用

概率论是研究和预测随机现象规律的数学分支，在竞技博弈中有广泛应用。

## 核心方法

### 基础概率
- **古典概率**：P(A) = 有利结果数 / 总结果数（掷骰子、抽签）
- **频率概率**：大量重复试验的频率趋近概率（大数定律）
- **贝叶斯概率**：根据新信息更新先验概率
  - `P(A|B) = P(B|A)·P(A) / P(B)`

### 推演工具
- **条件概率与全概率公式**：分解复杂事件为互斥子事件
- **贝叶斯推断**：从结果反推原因（医学检测、垃圾邮件过滤）
- **期望值 E(X)**：加权平均，预测长期平均结果
- **方差 & 标准差**：衡量不确定性/风险

### 概率分布模型

| 分布 | 适用场景 |
|------|---------|
| 二项分布 | n次独立重复试验的成功次数 |
| 泊松分布 | 单位时间内随机事件发生次数 |
| 正态分布 | 大量独立随机因素叠加的结果 |
| 指数分布 | 等待时间、寿命分析 |
| 均匀分布 | 等可能区间内的取值 |

### 高级方法
- **蒙特卡洛模拟**：大量随机抽样模拟复杂情景
- **马尔可夫链**：基于当前状态预测下一步
- **中心极限定理**：大量独立变量之和近似正态分布

## 经典博弈场景

### 德州扑克
核心是**胜率 vs 赔率**的决策：
- 计算听牌概率（如转牌+河牌击中同花的概率 ≈ 34.9%）
- 底池赔率决策：胜率 > 底池赔率要求 → 跟注
- 隐含赔率：考虑击中后的额外收益

### 体育博彩
- **期望值分析**：EV = Σ(Pi × Vi)，选择 EV > 0 的投注
- **凯利公式**：`f* = (bp - q) / b`，确定最优投注比例

### 石头剪刀布
- **纳什均衡**：各出 1/3 概率
- **实战偏离**：利用人类模式偏好（新手偏爱石头，赢者倾向于重复）

### 21点
- **基本策略表**：根据手牌和庄家明牌决定要牌/停牌/加倍
- **高低法计数**：2-6 为 +1，10-A 为 -1，根据真实计数调整下注

### MOBA 团战决策
- 决策树分析：视野、复活时间、经济差等因素量化
- 蒙特卡洛模拟比赛结果

## 核心思维框架

```
1. 识别所有可能结果
2. 估算每种结果的概率
3. 计算期望值 EV = Σ(Pi × Vi)
4. 选择 EV > 0 的决策
5. 用凯利公式控制风险
6. 持续用贝叶斯更新概率
```

> 🎲 **关键原则**：概率论不保证单次结果，但保证长期重复下的最优表现。短期有方差，长期看期望。
