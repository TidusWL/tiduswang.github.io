# 纳什均衡

> **Stub 页**：仅作为概率博弈知识体系的占位节点，详细内容待补充。

## 简要定义
**纳什均衡**（Nash Equilibrium）：在一个 n 人博弈中，若给定其他参与者的策略不变，任何单个参与者都无法通过单方面改变自己的策略来获得更高收益，则当前的策略组合构成纳什均衡。

## 在石头剪刀布中的应用
- 纯策略：各出 1/3 概率（无纯策略纳什均衡）
- 实战偏离：人类模式偏好（新手偏爱石头、赢者倾向于重复）→ 存在可被利用的"非均衡"漏洞
