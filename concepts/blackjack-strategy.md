# 21点基本策略与算牌法

21点（Blackjack）是赌场中玩家优势最高的游戏之一，概率论可以显著提升胜率。

## 基本策略表

根据手牌和庄家明牌的最优决策：

| 你的手牌 | 庄家明牌 2-6 | 庄家明牌 7-A |
|---------|-------------|------------|
| 12-16 | 停牌 | 要牌 |
| 11 | 加倍 | 加倍 |
| 10 | 加倍（除9外） | 要牌 |
| 9 | 加倍（3-6） | 要牌 |
| A,7 | 加倍 | 要牌 |
| A,6 | 加倍（3-6） | 要牌 |
| 对A/8 | 分牌 | 分牌 |
| 对10 | 不分 | 不分 |
| 对5 | 加倍 | 要牌 |

> 庄家明牌 2-6 为"弱牌"（容易爆牌），7-A 为"强牌"

## 高低法算牌（Hi-Lo Count）

### 计分规则
- **2-6 出现**：+1（小牌出完对玩家有利）
- **10-A 出现**：-1（大牌出完对庄家有利）
- **7-9 出现**：0（中性牌）

### 真实计数
```
真实计数 = 高低计数 / 剩余牌堆数
```

### 策略调整

| 真实计数 | 优势变化 | 建议 |
|---------|---------|------|
| ≤0 | 庄家优势 0.5% | 最小注 |
| +1 | 接近公平 | 正常下注 |
| +2 | 玩家优势 0.5% | 加注 2x |
| +3 | 玩家优势 1% | 加注 3x |
| +5 | 玩家优势 2% | 加注 5x |

## 为什么算牌有效？

- 大牌（10-A）多 → 玩家更容易拿到 21 点，庄家更容易爆牌
- 小牌多 → 庄家不容易爆牌，玩家难以改善手牌
- 6 副牌中移除一张 10，玩家优势变化约 0.5%

## 赌场反制措施

- 增加牌副数（6-8 副）
- 频繁洗牌
- 禁止算牌者入场
- 使用 CSM（连续洗牌机）
